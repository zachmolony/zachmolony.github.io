Wabiyo.com

Context

Wabiyo.com is a site and clothing store I built, as a digital platform for creative collective and clothing company WABIYO, of which I am a co-founder. All source code is on Github.

Desktop JS Game

Unlike a normal store, the store is a platformer game, coded in javascript using the p5 module, where the player/customer can traverse a dungeon to find the store and check out their items.

Mobile VR

The mobile version of the store uses a discreet design, but the background of the store is a Virtual Reality scene of a "psychedelic space", programmed and implemented using the A-FRAME framework. Try it yourself at wabiyo.com.

DARK LAB Studios

DARK LAB

I was hired to design and implement a website for fashion brand DARK LAB following my work on WABIYO. The site is currently still being developed to handle clothing drops, secure payment and email sign ups. A desktop version is still in development.

In this project I taught myself to implement animations using JQuery and javascript with CSS, amd to improve my markup and design skills This was an especially interesting process working for a client; creating prototypes and designs, responding to client feedback and adapting to changing requirements was something I found exciting.
