Wabiyo

Website

Context

Wabiyo.com is a site I built completely from scratch, as a digital platform for creative collective and clothing company wabiyo, of which I am a co-founder. All source code is on Github and hosted with Github Pages.

Desktop JS Game

Unlike a normal store, the store is a dungeon platformer game, coded in javascript using the p5 module, where the player/customer can play through the short game to find the item they want and check out.

Mobile VR

The mobile version of the store uses a simple design, but the background of the store is a Virtual Reality scene of a "psychedelic space", programmed using the A-FRAME framework. Try it yourself at wabiyo.com.
